# Contributing to Modudraft

Thanks for helping make Modudraft better!

## Bugs and feature requests

Use [GitHub Issues](https://github.com/modudraft/feedback/issues/new/choose) — pick the right template.

## Questions

Use [GitHub Discussions](https://github.com/modudraft/feedback/discussions) → Q&A category.

## Security vulnerabilities

Email [security@modudraft.com](mailto:security@modudraft.com) — do not file a public issue.

## What this repo is

This is the public feedback repository. Modudraft is closed source. We track bugs, feature requests, and community discussions here.
