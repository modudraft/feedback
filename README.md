# Modudraft — Feedback & Issues

This is the public issue tracker for [Modudraft](https://modudraft.com) — the architecture
diagramming tool for engineering teams.

The main codebase is private, but we use this repo for bugs, feature requests, and questions.

## Filing a bug

Open an [issue](https://github.com/modudraft/feedback/issues/new) and include:

- What you were doing
- What you expected to happen
- What actually happened
- Browser and OS (if relevant)

## Requesting a feature

Open an [issue](https://github.com/modudh a short description
of the problem you're trying to solve. We read every request and use GitHub reactions (👍) to
prioritize.

## Questions

Open an issue or email [hello@modudraft.om).

## Security issues

Please do **not** file security vulnerab
Email [security@modudraft.com](mailto:security@modudraft.com) instead.
